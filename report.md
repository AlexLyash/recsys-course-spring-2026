## Homework 2 Report

### Abstract

В данной работе реализована рекомендательная система на основе модели BERT4Rec (RecTools) для генерации item-to-item рекомендаций музыкальных треков. Для формирования обучающих последовательностей используется правило отбора взаимодействий на основе статистик распределения времени прослушивания. Дополнительно применяется постобработка рекомендаций с удалением треков того же исполнителя, что позволяет повысить разнообразие выдачи.



### Детали

Данные представляют собой взаимодействия пользователь–трек с величиной `time ∈ [0,1]`, отражающей долю прослушивания трека. Для выделения значимых взаимодействий используется статистическое правило:

[
time \ge \mu + \frac{\sigma}{2}
]

где:

* (\mu) — среднее значение `time` по датасету
* (\sigma) — стандартное отклонение

Такой порог позволяет учитывать разброс значений и выделять взаимодействия выше среднего уровня вовлеченности без чрезмерной жесткости фильтрации.

После фильтрации формируются пользовательские последовательности фиксированной длины, которые подаются в модель BERT4Rec. В обучении также используются признаки треков (например, исполнитель и жанр).

После обучения вычисляются item-to-item рекомендации. На этапе постобработки:

* удаляются треки того же исполнителя,
* из оставшихся выбираются top-K рекомендаций.
